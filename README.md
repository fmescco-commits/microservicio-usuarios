# Microservicio de Usuarios

Este proyecto corresponde al microservicio de usuarios del sistema de comercio electrónico basado en microservicios.

## Tecnologías
- Lenguaje / Framework: (colocar)
- Base de datos: MongoDB
- Arquitectura: API REST
- Contenedores: Docker

## Ejecución
El servicio se ejecuta como parte del entorno Docker Compose definido en el proyecto principal.
